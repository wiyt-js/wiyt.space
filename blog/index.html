Hey there!

As we enter my birthday month, I updated even more functionality to the Blog page today. This update includes a neat little comments system called Giscus. It grants users with GitHub accounts the ability to comment and react on posts and pages.

Giscus is basically the cooler, GitHub version of Disqus-- except it's open-source (rather than having their third-party servers store your discussions) and more lightweight. Pretty cool how it utilizes the Discussions feature built into the repository for my site directly. No ads and tracking is always an amazing plus too!

If you happen to have a GitHub account, you can try commenting on this main Blog page below!
